# アセットファイル配置手順

## 📁 ファイル配置場所

Google Driveからダウンロードしたファイルを、以下のフォルダに配置してください：

```
rissai-avatar-generator/
└── public/
    └── assets/
        ├── face/          # 顔のパーツ
        ├── hair/          # 髪型のパーツ
        ├── mouth/         # 口のパーツ
        ├── eyes/          # 目のパーツ
        └── cheeks/        # 頬のパーツ
```

## 📝 ファイル命名規則

### 必須：以下の命名規則に従ってください

#### 髪型（hair/）
- `hair_1.png` または `hair_1.svg`
- `hair_2.png` または `hair_2.svg`
- ...
- `hair_10.png` または `hair_10.svg`

#### 目（eyes/）
- `eyes_1.png` または `eyes_1.svg`
- `eyes_2.png` または `eyes_2.svg`
- ...
- `eyes_10.png` または `eyes_10.svg`

#### 口（mouth/）
- `mouth_1.png` または `mouth_1.svg`
- `mouth_2.png` または `mouth_2.svg`
- ...
- `mouth_10.png` または `mouth_10.svg`

#### 頬（cheeks/）
- `blush.png` または `blush.svg` （赤らめるパターン）
- `no_blush.png` または `no_blush.svg` （赤らめないパターン）

#### 顔（face/）
- `face_base.png` または `face_base.svg` （顔のベース）

## 🔄 配置手順

1. **Google Driveからファイルをダウンロード**
   - Google Driveのフォルダを開く
   - 各フォルダ（顔、髪型、口元、目、頬）のファイルをダウンロード

2. **ファイル名を確認・変更**
   - 上記の命名規則に従っているか確認
   - 必要に応じてファイル名を変更

3. **プロジェクトに配置**
   - ダウンロードしたファイルを、対応する `public/assets/` のサブフォルダに配置
   - 例：髪型のファイル → `public/assets/hair/` に配置

## ⚠️ 注意事項

- **ファイル名は正確に一致させる必要があります**（大文字小文字も区別されます）
- **SVG形式を推奨**（拡大縮小しても品質が保たれます）
- PNG形式も対応可能です
- **囲み（食べ物等）は後で手書きで用意するため、今は配置不要です**

## ✅ 配置完了後の確認

ファイルを配置したら、開発サーバーを再起動してください：

```bash
npm run dev
```

ブラウザで確認し、画像が正しく表示されることを確認してください。

## 🎨 髪色について

髪色は画像ファイルではなく、コード内で色を適用する想定です。
もし髪色ごとに別ファイルがある場合は、以下の命名規則で配置できます：

- `hair_1_color_1.png`, `hair_1_color_2.png`, `hair_1_color_3.png` など

この場合は、コードの修正が必要になります。まずは基本の `hair_1.png` などで配置して、必要に応じて対応します。
